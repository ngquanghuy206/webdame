📖 Hướng dẫn sử dụng FB Dame Tool
🚀 Cách sử dụng
1

Lấy Cookie Facebook

Mở Facebook.com trên Chrome/Edge → F12 → Tab Application → Cookies → copy toàn bộ.
Hoặc dùng extension EditThisCookie / Cookie-Editor để export dạng string.

💡 Cookie phải từ tài khoản Facebook còn sống, chưa bị checkpoint.
2

Dán Cookie vào ô "Cookie Facebook"

Paste cookie string vào ô đầu tiên. Bấm ✅ Kiểm tra để xác nhận cookie còn sống và lấy tên tài khoản.

3

Nhập link / UID target

Dán link profile Facebook cần dame, ví dụ:
https://www.facebook.com/username
hoặc nhập UID số trực tiếp.

4

Chọn tốc độ và bấm Bắt đầu

Chọn tốc độ phù hợp rồi bấm 🐬 Bắt đầu Dame. Theo dõi trạng thái ở tab 📊 Trạng thái.

💡 Tốc độ "Chậm" ít bị rate limit hơn, dùng khi cookie yếu.
